## Technical Context

### Technology Stack

**Core Technologies:**
- **Language**: Kotlin 1.9.20
- **Build System**: Gradle 8.x with Kotlin DSL
- **Minimum SDK**: API 26 (Android 8.0) - 98.8% market coverage
- **Target SDK**: API 34 (Android 14)

**UI Framework:**
- **Jetpack Compose**: 1.9.3 via BOM 2025.10.00
- **Material Design 3**: Latest with dynamic color support
- **Compose Navigation**: Type-safe routing

**Architecture Components:**
- **Dependency Injection**: Hilt 2.48+
- **Database**: Room 2.6+ with KSP annotation processing
- **State Management**: StateFlow + ViewModel (MVI pattern)
- **Async**: Kotlin Coroutines

**Code Quality:**
- **Linting**: ktlint 11.6.1 with Android ruleset
- **Testing**: JUnit 4, Kotlin Test, Compose UI Test

**Build Configuration:**
- Kotlin DSL for all Gradle files
- KSP for annotation processing (2x faster than kapt)
- R8 shrinking for release builds
- Gradle build caching enabled

### Key Architectural Decisions

1. **Compose BOM 2025.10.00**: Ensures version consistency across all Compose libraries
2. **Hilt over Dagger/Koin**: Compile-time safety with Android optimization
3. **KSP over KAPT**: 2x faster annotation processing
4. **MVI Pattern**: Unidirectional data flow with StateFlow
5. **Feature-first packages**: Scalable project structure
6. **Material 3**: Modern design with dynamic theming

See [research.md](./research.md) for detailed technical research and alternatives considered.
